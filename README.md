# Brick Blast

A turn-based brick breaker game for Android built with Kotlin and Jetpack Compose.

## Features

- Turn-based gameplay with trajectory preview
- Custom 60 FPS physics engine with spatial hash collision detection
- Particle effects with object pooling
- Power-ups: Multi Ball, Bomb, Laser, Slow Motion, Double Damage
- Game modes: Classic, Challenge, Time Attack, Hardcore
- Room persistence for coins, upgrades, achievements, statistics, and auto-save

## Build

Open in Android Studio (API 26+) or run:

```bash
./gradlew assembleDebug
```
